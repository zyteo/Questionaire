# Questionaire

MERN Questionaire.


### Technology Used

Technologies used to build this project.

```
- MongoDB
- Express
- ReactJS + React router
- NodeJS
- Styled components
- bcrypt
- JWT
- mongoose

```

### User Stories

```
To use the app, user should:

- Create an account (default role is user)
- Log in
- Answer Question

```
